Components 

-> Page -> Components -> Atoms

1. Page: A page is a complete screen in the app, such as a home page or a sign-in page. It is made up of multiple components and represents a specific view in the app.
2. Components: Components are reusable building blocks that can be used across different pages. They can be made up of smaller components or atoms. Examples of components include buttons, input fields, and cards.
3. Atoms: Atoms are the smallest building blocks in component-driven design. They are simple, indivisible elements that cannot be broken down further. Examples of atoms include text, icons, and colors.
